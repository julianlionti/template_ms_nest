## Description

[Nest](https://github.com/nestjs/nest) with Microservices, monorepo, docker and development watch

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:all


# docker mode
$ docker-compose up -d
```
